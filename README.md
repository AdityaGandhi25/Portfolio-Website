Welcome to the repository for **Aditya Gandhi's** personal portfolio website! This project is a culmination of creativity, technology, and dedication to showcasing skills, projects, and personality in a visually stunning and interactive manner.

## ✨ Key Features

- **3D Interactive Animations**  
  Features a custom-made interactive keyboard built with Spline. Each keycap represents a data/tech skill and reveals titles and descriptions on hover.
  
- **Slick Interactions & Animations**  
  Powered by GSAP and Framer Motion, delivering buttery-smooth animations on scroll, hover, and element reveals.

- **Space-Themed Design**  
  Particles floating on a dark, cosmic background simulate an outer-space vibe, giving a unique and futuristic look.

- **Fully Responsive**  
  Layout ensures the website looks and functions beautifully on desktop and mobile.

## 🛠️ Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS, Shadcn, Aceternity UI
- **Animations:** GSAP, Framer Motion, Spline Runtime
- **Other Tools:** Resend, Socket.io, Zod

## 🚀 Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/AdityaGandhi25/3D-interactive-portfolio.git
   cd 3D-interactive-portfolio
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   Create a `.env.local` file and add your Resend API key:
   ```env
   RESEND_API_KEY=your_resend_api_key_here
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

*Note: The `next.config.mjs` file contains an experimental ngrok link in the `allowedDevHosts` configuration. You should replace this link with your own ngrok domain, or remove the `experimental` block entirely if you are running locally without a tunnel.*

## 🌐 Deployment

This project is deployed using **Vercel**. To set up the Resend API on Vercel:
1. Sign up for a [Resend](https://resend.com) account and get your API key.
2. Go to your Vercel project settings > Environment Variables.
3. Add `RESEND_API_KEY` with your key and redeploy.
